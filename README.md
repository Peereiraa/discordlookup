Discord Lookup es un proyecto frontend en HTML, CSS y JavaScript que permite:

Buscar usuarios de Discord y mostrar su avatar, banner, edad de la cuenta, conexiones y más detalles técnicos.

Consultar servidores (guilds) de Discord mostrando su icono, descripción, número de miembros, fecha de creación, nivel de verificación, suscripciones y características destacadas.

Presentar la información de manera visual en tarjetas responsivas, con colores, badges y meta datos relevantes.

Soporte para diferentes tipos de datos (usuarios o servidores) desde una sola interfaz.

Ideal para administradores de servidores, desarrolladores de bots o simplemente curiosos que quieren explorar información de Discord de forma rápida y organizada.

Tecnologías:

HTML5 / CSS3 / JavaScript

Fetch API para consumir endpoints locales de usuario y servidor
